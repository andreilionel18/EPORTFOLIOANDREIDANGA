Lesson 2: Types of CSS


  • HOW CSS IS APPLIED

    • Linked method
    CSS is stored in a separate file, instead of directly in the HTML page
    Embedded method
    CSS is stored as part of the HTML page, in the header section.
    Inline method
    CSS is stored directly in the style attributes of the HTML tags

    Inline style
    We can define the style for a single element using the style attribute.
    There are various tags that have the style attribute and that said line would be its value.
    Embedded style
    "style type=”text/css" "/style"
    placed within the head part of the HTML file
    provides style to the WHOLE html file

    Linked style
    made outside the html file
    uses the "link" element, it allows you to establish document relationships and tells the browser to find the specified style sheet
    it can only be used within the "head""/head" section of the document
    the style sheet file does not contain any HTML code, it contains only style rules

    Creating an External CSS

    Open a blank document in Notepad.
    Type the style you want to define.
    On the Menu Bar, click File and click Save.
    Type your filename with a file extension .css and change the file type to all files.
